# REMtracker

There are 2 main software packages:
- miniscope
- remtracker

Package "miniscope" contains the following classes:
- Miniscope
- MiniScopeDataModel
- MiniScopeTimerTask
- Channel
- ChannelFunctions
- MiniScopeEnum
- MiniScopeEventRelayer

Package "remtracker" contains the following classes:
- REMtracker
- REMtrackerDataModel
- REMtrackerPane
- WiFiHandler
- SerialHandler
